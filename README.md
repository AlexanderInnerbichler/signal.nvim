# signal.nvim

Signal messenger inside Neovim. Needs [signal-cli](https://github.com/AsamK/signal-cli) and Java 25+.

## Install

### 1. Install Java 25

```bash
sudo apt install openjdk-25-jre-headless
```

### 2. Install signal-cli

Download v0.14.x from the [releases page](https://github.com/AsamK/signal-cli/releases):

```bash
tar -xf signal-cli-0.14.3.tar.gz
mv signal-cli-0.14.3 ~/signal-cli-0.14.3
ln -sf ~/signal-cli-0.14.3/bin/signal-cli ~/.local/bin/signal-cli
```

### 3. Add to Neovim

```lua
{
  dir = "~/signal.nvim",
  lazy = false,
  config = function()
    require("signal").setup()
  end,
}
```

Add a keymap:

```lua
vim.keymap.set("n", "<leader>si", function() require("signal").toggle() end,
  { desc = "Toggle Signal" })
```

### 4. Link your device

Open Neovim and run:

```
:SignalLink
```

A QR code URL is printed — open it in your browser, scan from Signal on your phone
(**Settings → Linked devices → Link new device**). Your account is cached after a
successful link; no re-linking on next start.

> First time registering a brand-new number instead of linking? Use `:SignalSetup`.

## Config

```lua
require("signal").setup({
  poll_interval = 30,           -- seconds between background polls
  notif_ttl     = 5,            -- toast auto-dismiss seconds
  window_width  = 0.9,          -- floating window width fraction
  signal_cmd    = "signal-cli", -- path to signal-cli binary
})
```

## Keys

### Conversation list

| Key | Action |
|-----|--------|
| `<CR>` | Open thread |
| `n` | New chat (pick from contacts) |
| `/` | Filter conversations by name |
| `<Esc>` | Clear filter / close |
| `r` | Refresh |
| `p` | View contact profile |
| `P` | Pin / unpin conversation |
| `q` | Close |

### Thread view

| Key | Action |
|-----|--------|
| `s` | Compose new message |
| `gr` | Reply (quotes selected message) |
| `ra` | React with emoji |
| `rd` | Delete message for everyone |
| `r` | Refresh |
| `p` | View contact profile |
| `q` / `<Esc>` | Back to list |
