# signal.nvim

Signal messenger inside Neovim. Needs [signal-cli](https://github.com/AsamK/signal-cli) and Java 25+.

## Install

### 1. Install Java 25

```bash
sudo apt install openjdk-25-jre-headless
```

### 2. Install signal-cli

Download v0.14.x from the [releases page](https://github.com/AsamK/signal-cli/releases):

```bash
tar -xf signal-cli-0.14.3.tar.gz
mv signal-cli-0.14.3 ~/signal-cli-0.14.3
ln -sf ~/signal-cli-0.14.3/bin/signal-cli ~/.local/bin/signal-cli
```

### 3. Add to Neovim

```lua
{
  dir = "~/signal.nvim",
  lazy = false,
  config = function()
    require("signal").setup()
  end,
}
```

Add a keymap:

```lua
vim.keymap.set("n", "<leader>si", function() require("signal").toggle() end,
  { desc = "Toggle Signal" })
```

### 4. Register your number

```
:SignalSetup
```

The wizard handles registration, captcha, and SMS verification entirely inside Neovim.
Your number is cached after a successful verify — no re-registration on next start.

## Config

```lua
require("signal").setup({
  poll_interval = 30,           -- seconds between background polls
  notif_ttl     = 5,            -- toast auto-dismiss seconds
  window_width  = 0.9,          -- floating window width fraction
  signal_cmd    = "signal-cli", -- path to signal-cli binary
})
```

## Keys

### Conversation list

| Key | Action |
|-----|--------|
| `<CR>` | Open thread |
| `/` | Filter conversations by name |
| `r` | Refresh |
| `<Esc>` | Clear filter / close |
| `q` | Close |

### Thread view

| Key | Action |
|-----|--------|
| `s` | Compose message |
| `r` | Refresh |
| `q` / `<Esc>` | Back to list |

### Compose popup

| Key | Action |
|-----|--------|
| `<C-s>` | Send |
| `q` / `<Esc>` | Cancel |

### Setup wizard

| Key | Action |
|-----|--------|
| `<CR>` / `<C-s>` | Submit |
| `q` | Cancel |

`:checkhealth signal` to verify setup.
